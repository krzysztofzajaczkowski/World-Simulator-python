# Turn based world simulator in Python

Turn based world simulator for Object Oriented Programming project at GUT

## GUI module used

Tkinter

## Key bindings

- Arrow Up
  - Move upwards
- Arrow Right
  - Move right
- Arrow Down
  - Move down
- Arrow Left
  - Move left
- R
  - Activate super skill
- Space
  - End turn
